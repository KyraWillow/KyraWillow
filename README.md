## about this journey

I am currently building a stronger foundation in the data field — slowly, seriously, and through real practice.

What interests me most is not only the result, but the process behind it:  
how raw data is cleaned, transformed, structured, and turned into something useful.

Right now, I am focused on:

- SQL fundamentals and query thinking  
- Python for data work  
- data cleaning and transformation  
- simple ETL workflows  
- building portfolio-ready projects  

<br>
